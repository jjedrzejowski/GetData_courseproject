This file shows the variables used in the final set. More detailed description can be found in the original data, as file "features_info.txt"

ID variables:

Activity - six different levels: 
1 WALKING

2 WALKING_UPSTAIRS

3 WALKING_DOWNSTAIRS

4 SITTING

5 STANDING

6 LAYING
